## 2. 전체 실행 흐름

```
run.bat
  └─ python main.py
       └─ QApplication + MainWindow (PySide6 GUI)
            └─ [실행 버튼] → OrchestratorWorker (QThread)
                 ├─ _preflight()            # 저장 경로 검증
                 ├─ kill_chrome_processes() # 기존 Chrome 종료 (taskkill)
                 ├─ mklink /J junction udd  # ← 핵심 우회 로직 (아래 §3 참조)
                 ├─ subprocess.Popen(chrome.exe --remote-debugging-port=9222)
                 ├─ 포트 9222 폴링 대기 (최대 20초)
                 ├─ playwright.connect_over_cdp("http://127.0.0.1:9222")
                 └─ _run_all()
                      └─ 각 (매체, 계정) 쌍:
                           downloader.run(context, start, end)
                             ├─ context.new_page()      # 새 탭 생성
                             ├─ check_login(page)
                             ├─ navigate_to_report(page)
                             ├─ set_period(page, start, end)
                             ├─ trigger_download(page, start, end) → tmp_path
                             ├─ build_destination_path() + move_download()
                             └─ page.close()
```

## 3. Chrome 실행 방식 및 현재 이슈

### 배경
- 사용자 Chrome 프로필(`Profile 4`)에 각 매체 로그인 세션이 저장되어 있음
- 자동화는 이 세션을 재사용해야 함 (ID/PW 코드 저장 금지 정책)

### 시도한 방법들과 실패 원인

| 방법 | 실패 원인 |
|---|---|
| `playwright.launch_persistent_context(channel="chrome")` | Chrome 기본 User Data Dir에서 `--remote-debugging-pipe` 차단: *"DevTools remote debugging requires a non-default data directory"* |
| `subprocess.Popen(chrome.exe --remote-debugging-port=9222, user-data-dir=기본경로)` | 기본 User Data Dir에서 `--remote-debugging-port`도 차단 → 포트 9222가 열리지 않음 |

### 현재 시도 중: Directory Junction 우회

```python
# Chrome이 "기본 경로"로 인식하지 못하도록 Junction(심볼릭 링크)을 만들어 우회

    chrome_exe,

    ...

```

**검증 필요**: 이 방법이 Chrome의 기본 경로 감지를 실제로 우회하는지 확인 필요

```python
class BaseDownloader(ABC):
    MEDIA_CODE: str = ""
    IMPLEMENTED: bool = False

    def __init__(self, config: dict, account: dict | None = None):
        self.config = config
        self.account: dict = account or {}       # ← 이번에 추가한 self.account 저장
        _media_cfg = config.get("media", {}).get(self.MEDIA_CODE, {})
        _acct = self.account
        self.timeout_sec  = _media_cfg.get("timeout_sec", 90)
        self.report_name  = (_acct.get("report_name") or _media_cfg.get("report_name", "")).strip()
        self.account_id   = (_acct.get("account_id") or "").strip()
        self.account_name = (_acct.get("account_name") or self.account_id or "").strip()
        self.max_attempts = config.get("retry", {}).get("max_attempts", 2)
        self.save_root    = config.get("save_root_path", "")

    def run(self, context, start_date, end_date) -> DownloadResult:
        for attempt in range(1, self.max_attempts + 1):
            page = context.new_page()
            try:
                if not self.check_login(page):
                    raise LoginRequiredError()
                self.navigate_to_report(page)
                self.set_period(page, start_date, end_date)
                tmp_path = self.trigger_download(page, start_date, end_date)
                dest = build_destination_path(...)
                final = move_download(tmp_path, dest)
                return DownloadResult(success=True, dest_path=final)
            except LoginRequiredError:
                return self._skip("로그인 필요", login_required=True)
            except Exception as e:
                # retry ...
            finally:
                page.close()

    @abstractmethod
    def check_login(self, page) -> bool: ...
    @abstractmethod
    def navigate_to_report(self, page) -> None: ...
    @abstractmethod
    def set_period(self, page, start, end) -> None: ...
    @abstractmethod
    def trigger_download(self, page, start, end) -> Path: ...
```

## 7. 현재 미해결 이슈 (검토 요청)

### 이슈 1: Chrome 원격 디버깅 포트가 열리지 않음 ⚠️ **주요 블로커**

**증상**:
```
[11:16:57] [SYSTEM] ✗ RuntimeError: Chrome 디버깅 포트(9222)가 열리지 않았습니다.
```

**원인**: Chrome은 기본 User Data Dir(`%LOCALAPPDATA%\Google\Chrome\User Data`)에서  
`--remote-debugging-pipe` 및 `--remote-debugging-port` 모두 차단

**현재 시도**: Directory Junction으로 다른 경로처럼 우회 (검증 중)

**질문**:
1. Junction 방식이 실제로 Chrome의 기본 경로 감지를 우회하는가?
2. 더 나은 대안이 있는가?
   - 별도 User Data Dir 생성 후 Profile 복사?
   - Chrome 실행 후 `--restore-last-session` 조합?
   - Selenium ChromeDriver의 `debuggerAddress` 옵션?

### 이슈 2: connect_over_cdp 후 context.new_page() 동작 확인 필요

```python
browser = pw.chromium.connect_over_cdp("http://127.0.0.1:9222")
context = browser.contexts[0]   # 기존 세션 컨텍스트
page = context.new_page()        # 새 탭 — 세션 쿠키 공유되는가?
```

**질문**: `connect_over_cdp`로 얻은 `context`에서 `new_page()`를 호출하면  
기존 로그인 세션(쿠키)이 새 탭에 유지되는가?

### 이슈 3: 매체별 셀렉터 미검증

실제 브라우저에서 동작을 아직 확인하지 못한 매체:
- Meta: `data-surface='am/lib:export_button'` 셀렉터
- X (Twitter): 내보내기 다이얼로그 전체 흐름
- Mobion: Chrome Password Manager autofill 시나리오

## 8. 파일별 전체 코드

### core/orchestrator.py

```python
"""
OrchestratorWorker — QThread that runs all media downloaders sequentially.
"""
from __future__ import annotations
from datetime import date
from typing import TYPE_CHECKING

from PySide6.QtCore import QThread, Signal
from config_schema import MEDIA_ORDER
import utils.logger as log_mod

if TYPE_CHECKING:
    from downloader.base import BaseDownloader, DownloadResult


class RunParams:
    def __init__(self, start_date, end_date, enabled_media, config, use_mock=True):
        self.start_date = start_date
        self.end_date = end_date
        self.enabled_media = enabled_media
        self.config = config
        self.use_mock = use_mock


def _get_accounts(media_cfg: dict) -> list[dict]:
    accounts = media_cfg.get("accounts")
    if accounts:
        return list(accounts)
    return [{"account_id": media_cfg.get("account_id", "").strip(),
             "account_name": "",
             "report_name": media_cfg.get("report_name", "")}]


def _build_downloader(media_code, config, account, use_mock):
    from downloader.mock import MockDownloader
    if use_mock:
        return MockDownloader(media_code, config, account)

    _real_cls = None
    try:
        if media_code == "naver":
            from downloader.naver import NaverDownloader as _real_cls
        elif media_code == "google":
            from downloader.google import GoogleDownloader as _real_cls
        elif media_code == "meta":
            from downloader.meta import MetaDownloader as _real_cls
        elif media_code == "kakao":
            from downloader.kakao import KakaoDownloader as _real_cls
        elif media_code == "adn":
            from downloader.adn import AdnDownloader as _real_cls
        elif media_code == "mobion_banner":
            from downloader.mobion_banner import MobionBannerDownloader as _real_cls
        elif media_code == "mobion_daily":
            from downloader.mobion_daily import MobionDailyDownloader as _real_cls
        elif media_code == "x":
            from downloader.x_ads import XAdsDownloader as _real_cls
    except ImportError:
        pass

    if _real_cls is not None and getattr(_real_cls, "IMPLEMENTED", False):
        return _real_cls(config, account)

    log_mod.log_warning(media_code, "실제 다운로더 미구현 — 테스트 모드로 실행")
    return MockDownloader(media_code, config, account)


class OrchestratorWorker(QThread):
    log_message          = Signal(str, str, str)
    media_status_changed = Signal(str, str)
    progress_updated     = Signal(int, int)
    run_finished         = Signal(dict)
    login_required       = Signal(str)

    def __init__(self, params: RunParams, parent=None):
        super().__init__(parent)
        self._params = params
        self._stop_requested = False
        self._current = None

    def request_stop(self):
        self._stop_requested = True
        if self._current:
            self._current.request_stop()

    def run(self):
        params = self._params
        enabled = [m for m in MEDIA_ORDER if m in params.enabled_media]
        summary = {}
        log_mod.set_ui_callback(self._emit_log)

        try:
            if not self._preflight(params, summary):
                self.run_finished.emit(summary)
                return

            if params.use_mock:
                self._run_all(None, enabled, summary, params)
            else:
                import os, subprocess, time as _time, tempfile
                from playwright.sync_api import sync_playwright
                from core.chrome_lock_checker import kill_chrome_processes
                import urllib.request as _urllib

                chrome = params.config.get("chrome", {})
                udd  = chrome.get("user_data_dir", "")
                prof = chrome.get("profile_directory", "Default")

                chrome_exe = self._find_chrome_exe()
                self._emit_log(self._ts(), "SYSTEM", f"Chrome 경로: {chrome_exe}")

                self._emit_log(self._ts(), "SYSTEM", "Chrome 프로세스 정리 중...")
                kill_chrome_processes(timeout_sec=2.0)

                # Junction으로 기본 경로 감지 우회
                junction = os.path.join(tempfile.gettempdir(), "ad_report_chrome_udd")
                subprocess.run(["cmd", "/c", "rmdir", junction], capture_output=True)
                subprocess.run(["cmd", "/c", "mklink", "/J", junction, udd], capture_output=True)
                self._emit_log(self._ts(), "SYSTEM", f"Junction 생성: {junction} → {udd}")

                self._emit_log(self._ts(), "SYSTEM", f"Chrome 실행 중... profile={prof}")
                chrome_proc = subprocess.Popen([
                    chrome_exe,
                    f"--user-data-dir={junction}",
                    f"--profile-directory={prof}",
                    "--remote-debugging-port=9222",
                    "--no-first-run",
                    "--no-default-browser-check",
                    "--disable-session-crashed-bubble",
                    "--disable-background-mode",
                ])

                for _i in range(20):
                    _time.sleep(1)
                    try:
                        _urllib.urlopen("http://127.0.0.1:9222/json/version", timeout=1)
                        self._emit_log(self._ts(), "SYSTEM", f"포트 9222 열림 ({_i+1}초 후)")
                        break
                    except Exception:
                        pass
                else:
                    raise RuntimeError("Chrome 디버깅 포트(9222)가 열리지 않았습니다.")

                with sync_playwright() as pw:
                    browser = pw.chromium.connect_over_cdp(
                        "http://127.0.0.1:9222", timeout=30_000)
                    context = browser.contexts[0] if browser.contexts else browser.new_context()
                    self._emit_log(self._ts(), "SYSTEM", "Chrome 연결 성공")
                    try:
                        self._run_all(context, enabled, summary, params)
                    finally:
                        try: browser.close()
                        except Exception: pass
                        try: chrome_proc.terminate()
                        except Exception: pass
                        subprocess.run(["cmd", "/c", "rmdir", junction], capture_output=True)

        except Exception as e:
            self._emit_log(self._ts(), "SYSTEM", f"✗ {type(e).__name__}: {e}")
            summary["SYSTEM"] = "error"
        finally:
            log_mod.set_ui_callback(None)

        self.run_finished.emit(summary)

    def _preflight(self, params, summary):
        if params.config.get("save_root_path") and not params.use_mock:
            from utils.file_manager import validate_save_root
            ok, err = validate_save_root(params.config["save_root_path"])
            if not ok:
                self._emit_log(self._ts(), "SYSTEM", f"✗ 저장 경로 오류: {err}")
                summary["SYSTEM"] = "save_path_error"
                return False
        return True

    def _run_all(self, context, enabled, summary, params):
        units = []
        for media_code in enabled:
            media_cfg = params.config.get("media", {}).get(media_code, {})
            for acct in _get_accounts(media_cfg):
                units.append((media_code, acct))

        total = len(units)
        self._emit_log(self._ts(), "SYSTEM",
                       f"총 {len(enabled)}개 매체 / {total}개 계정 실행 시작")

        media_results = {}
        media_total = {m: len(_get_accounts(params.config.get("media", {}).get(m, {})))
                       for m in enabled}

        for idx, (media_code, account) in enumerate(units):
            if self._stop_requested:
                summary.setdefault(media_code, "skipped")
                self.media_status_changed.emit(media_code, "skipped")
                self.progress_updated.emit(idx + 1, total)
                continue

            acct_label = account.get("account_name") or account.get("account_id") or "기본"
            if media_code not in media_results:
                self.media_status_changed.emit(media_code, "running")

            self._emit_log(self._ts(), media_code, f"─── 시작 [{acct_label}] ───")

            downloader = _build_downloader(media_code, params.config, account, params.use_mock)
            self._current = downloader
            result = downloader.run(context, params.start_date, params.end_date)
            self._current = None

            if result.login_required:
                self.login_required.emit(media_code)
                status = "skipped"
            elif result.skipped:
                status = "skipped"
            elif result.success:
                status = "success"
            else:
                status = "failed"

            media_results.setdefault(media_code, []).append(status)
            self._emit_log(self._ts(), media_code, f"완료 [{acct_label}]: {status}")
            self.progress_updated.emit(idx + 1, total)

            if len(media_results[media_code]) == media_total[media_code]:
                all_st = media_results[media_code]
                final_status = ("failed" if any(s == "failed" for s in all_st)
                                else "success" if all(s == "success" for s in all_st)
                                else "skipped")
                summary[media_code] = final_status
                self.media_status_changed.emit(media_code, final_status)

    def _emit_log(self, ts, media, message):
        self.log_message.emit(ts, media, message)

    @staticmethod
    def _ts():
        from datetime import datetime
        return datetime.now().strftime("%H:%M:%S")

    @staticmethod
    def _find_chrome_exe():
        import os
        candidates = [
            r"C:\Program Files\Google\Chrome\Application\chrome.exe",
            r"C:\Program Files (x86)\Google\Chrome\Application\chrome.exe",
            os.path.expandvars(r"%LOCALAPPDATA%\Google\Chrome\Application\chrome.exe"),
        ]
        for p in candidates:
            if os.path.exists(p):
                return p
        raise RuntimeError("Chrome 실행 파일을 찾을 수 없습니다.")
```

```python
class BaseDownloader(ABC):
    MEDIA_CODE: str = ""
    IMPLEMENTED: bool = False

    def __init__(self, config: dict, account: dict | None = None):
        self.config  = config
        self.account = account or {}           # ← 전체 account dict 보존
        _media_cfg   = config.get("media", {}).get(self.MEDIA_CODE, {})
        _acct        = self.account
        self.timeout_sec  = _media_cfg.get("timeout_sec", 90)
        self.report_name  = (_acct.get("report_name") or _media_cfg.get("report_name", "")).strip()
        self.account_id   = (_acct.get("account_id") or "").strip()
        self.account_name = (_acct.get("account_name") or self.account_id or "").strip()
        self.max_attempts = config.get("retry", {}).get("max_attempts", 2)
        self.save_root    = config.get("save_root_path", "")
        self._stop_requested = False

    def run(self, context, start_date, end_date) -> DownloadResult:
        for attempt in range(1, self.max_attempts + 1):
            page = None
            try:
                page = context.new_page()
                if not self.check_login(page):
                    raise LoginRequiredError()
                self.navigate_to_report(page)
                self.set_period(page, start_date, end_date)
                tmp_path = self.trigger_download(page, start_date, end_date)
                dest  = build_destination_path(self.save_root, self.MEDIA_CODE,
                                               start_date, end_date,
                                               tmp_path.suffix, account_name=self.account_name)
                final = move_download(tmp_path, dest, overwrite=self.overwrite)
                return DownloadResult(media_code=self.MEDIA_CODE, success=True, dest_path=final)
            except LoginRequiredError:
                return self._skip("로그인 필요", login_required=True)
            except EmptyDataError:
                return self._skip("데이터 없음")
            except Exception as e:
                if attempt < self.max_attempts:
                    time.sleep(random.uniform(2.0, 4.0))
            finally:
                if page:
                    try: page.close()
                    except Exception: pass
        return DownloadResult(media_code=self.MEDIA_CODE, error="재시도 후 최종 실패")
```
