키워드별 소진내역 가공 프로그램

0. 다운로드 주의
   GitHub의 Source code(zip)가 아니라 release assets의
   keyword-spend-processor_*.zip 파일을 다운로드해야 합니다.

1. 처음 1회
   install.bat 더블클릭
   한글 배치가 필요한 경우 설치.bat도 사용할 수 있습니다.

2. 매일 실행
   run.bat 더블클릭
   한글 배치가 필요한 경우 실행.bat도 사용할 수 있습니다.

3. 기본 흐름
   - 브랜드 선택
   - 다운로드 폴더, 템플릿, 로그인/다운로더 명령 확인
   - 업로드 CSV 생성 또는 전체 실행
   - raw 파일이 없으면 다운로더를 자동 실행하고 raw 파일 생성을 기다림
   - 결과 파일은 브랜드/날짜 폴더 기준으로 저장

4. 날짜 기준
   - 화~금 실행: 전일 데이터
   - 월요일 실행: 금/토/일 데이터
   - 공휴일/수동 기간은 프로그램의 수동 기간에 입력

5. 외부 프로그램 연결
   이 배포본은 로그인 봇/다운로더 봇을 bots\report-downloader 안에 포함합니다.
   새로 압축을 푼 경우 프로그램이 기본 명령을 자동으로 잡습니다.

   로그인 명령 기본값:
   bots\report-downloader\login.bat

   다운로더 명령 기본값:
   bots\report-downloader\run.bat

6. 규칙 수정
   rules.bat 실행 후 브랜드별 규칙을 수정
   한글 배치가 필요한 경우 규칙편집.bat도 사용할 수 있습니다.

주의
   Excel 템플릿 파일이 열려 있으면 저장이 실패하거나 이전 값이 보일 수 있습니다.
   결과 파일 확인 전에는 같은 결과 파일을 닫아 주세요.
