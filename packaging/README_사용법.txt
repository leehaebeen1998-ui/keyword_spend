키워드별 소진내역 가공 프로그램

1. 처음 1회
   설치.bat 더블클릭

2. 매일 실행
   실행.bat 더블클릭

3. 기본 흐름
   - 브랜드 선택
   - 다운로드 폴더, 템플릿, 로그인/다운로더 명령 확인
   - 업로드 CSV 생성 또는 전체 실행
   - raw 파일이 없으면 다운로더를 자동 실행하고 raw 파일 생성을 기다림
   - 결과 파일은 브랜드/날짜 폴더 기준으로 저장

4. 날짜 기준
   - 화~금 실행: 전일 데이터
   - 월요일 실행: 금/토/일 데이터
   - 공휴일/수동 기간은 프로그램의 수동 기간에 입력

5. 외부 프로그램 연결
   로그인 명령 예시:
   C:\report-downloader\로그인.bat

   다운로더 명령 예시:
   C:\report-downloader\run.bat

6. 규칙 수정
   규칙편집.bat 실행 후 브랜드별 규칙을 수정

주의
   Excel 템플릿 파일이 열려 있으면 저장이 실패하거나 이전 값이 보일 수 있습니다.
   결과 파일 확인 전에는 같은 결과 파일을 닫아 주세요.
